# JWT-Zcoder
JWT-Zcoder is a JWT Decoding Plugin for [Caido](https://caido.io/).

This plugin is currently a work in progress, built using the [React material UI Caido Starterkit](https://github.com/bebiksior/react-material-ui-caido-starterkit) by [Bebiksior](https://github.com/bebiksior), this plugin also draws inspiration and code snippets from Bebiksiorâ€™s [EventBetter](https://github.com/bebiksior/EvenBetter) extension.
